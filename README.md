Sobre o Projeto
O presente repositório contém o código-fonte do Nexus, uma aplicação web focada na gestão e interação no ambiente escolar. Este projeto acadêmico está sendo desenvolvido como parte das atividades de formação no Instituto J&F, com o objetivo principal de consolidar e aplicar na prática os conceitos de desenvolvimento backend, modelagem e estruturação de interfaces web.

Tecnologias e Ferramentas
A arquitetura e o desenvolvimento deste sistema baseiam-se nas seguintes tecnologias:

Java: Linguagem principal utilizada para a lógica de negócios e desenvolvimento do backend.

JSP (JavaServer Pages) & Servlets: Tecnologias empregadas para a construção de páginas web dinâmicas e o roteamento de requisições HTTP.

HTML & CSS: Estruturação e estilização da interface de usuário.

Maven: Gerenciamento de dependências e automação do processo de build (configurado via pom.xml).

Estrutura e Execução Local
Para configurar e executar o projeto em um ambiente de desenvolvimento local, siga as diretrizes abaixo:

Clonagem do Repositório:

Bash
git clone https://github.com/gaPeotta/Site-escola.git
Importação na IDE: Abra a pasta do projeto em sua IDE de preferência. O repositório já inclui o diretório .idea, facilitando a integração nativa com o IntelliJ IDEA.

Gerenciamento de Dependências: Certifique-se de que o Maven está corretamente configurado para realizar o download automático das dependências do projeto.

Configuração do Servidor: É necessário um container de servlets como o Apache Tomcat para hospedar a aplicação. O projeto já conta com definições prévias no diretório .smarttomcat para auxiliar na configuração.

Execução: Inicie o servidor de aplicação configurado e acesse o sistema através do seu navegador local (padrão: http://localhost:8080).
